用代码写简历记录

2024-05-22 2 min read

之前自己找工作时一直是使用其他工具写简历，这次自己因为一些原因又要开始找工作了，于是想能不能自己使用 React 写一个简历呢。为什么要用网页技术写简历，我觉得有以下的好处：

  • 可以使用Git 进行版本控制，可以清楚的追溯修改的历史。再也不用维护几十份历史简历文件了。
  • 针对不同岗位可以创建不同的分支，方便管理，比如针对岗位 A 我可以专门创建一个分支来突出自己的优势，当我学会一个新的技能可以更新到简历上面的时候，我可以将其快速 merge到其他分支。
  • 使用代码可以对简历样式实现更加精确的控制，可以实现像素级别的精度，如果使用其他排版工具，需要自己手动拖拽。
  • 可以实现一些动态逻辑，比如一键切换中英文版本，这样在一定程度上也可以减少维护两份简历的压力。
  • 可以直接通过网页向外呈现，由于这个本来就是网页应用，是一个 static page，所以可以直接通过 github pages 等工具部署上去，之后还可以与我的博客、社交工具联动。

想到以上的好处，自己决定捡起之前荒废已久的 React ，找了一个脚手架就开干了。说实话比较简单，因为只是做一个静态网站。最终经过一百多次提交后，实现了以上的目标。
